DOCUMENTS REQUIRED

Please remember to carry the required documents in original along with self-attested copies of each document on the day of Meet & Greet for verification.
Self-attested copies of the documents to be submitted are mentioned below.

Please remember to carry the

  • Original Birth Certificate with Date of Birth, Child and Parent’s Name
  • Applicant’s Aadhaar Card (Mandatory)
  • Child’s active Passport (Conditional)
  • Guardian’s Aadhar Card (Mandatory)
  • Residence Proof of either parent (Any 2 documents from the list given below) (Mandatory)
    • Utility Bill (Water/ Electricity/ Phone etc.)
    • Sale deed/ lease deed / rent agreement / possession letter
    • Bank statement or Passbook
    • Passport / Aadhaar card / Driving License
  • Transfer Certificate (for all grades except Grade I) (Mandatory)
  • Report card for previous grade (showing promotion to the class for which admission is sought). (Mandatory)
  • Proof of Alumnus or other siblings studying in Arya Gurukul (Any Branch) any one document from the list given below
    • Migration Certificate of class 10th
  • Transfer Certificate
  • Marksheet of the previous grade
  • Legal documents in case of separation / divorce / adoption / single parent (Conditional)
  • If adopted, document/ gazette/gazetted without documents. (Conditional)
  • Caste certificate, if (SC/ST/OBC/SBC/VSNT) (Conditional)

Note:All submitted documents must have the same name of the Child / Father / Mother along with Date of Birth. Any mismatch between the documents need to be authorized by First Class Magistrate through an affidavit or Govt. Gazette.
